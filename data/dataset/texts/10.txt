WEB Поликлиника: Лист нетрудоспособности

h1. Лист нетрудоспособности


Работа с ЛН осуществляется в модуле учета ЛН. Для открытия модуля выберите на главной странице пункт «Учет листков нетрудоспособности».
{{8874c62c-0a49-498d-b5d7-879fc41cdd18.png}}
 
После выбора этого пункта загрузится страница «Учет листков нетрудоспособности», на которой отображается 
список всех заведенных в медицинской организации ЛН и поля фильтра списка в верхней части страницы. 
По умолчанию список пустой. Для отображения полного списка листков нетрудоспособности необходимо нажать клавишу Enter или кнопку «Найти», при этом оставив пустыми все остальные поля фильтра. В полях фильтра можно задавать критерии, согласно которым можно фильтровать список ЛН.

{{d96ad917-ce74-4178-ad0f-c89f0758c381.png}}
 
Для загрузки листка нетрудоспособности выданного из другой МО нажать кнопку «Получить ЛН от ФСС». Откроется окно «Получение листка нетрудоспособности».
Параметрами запроса будет выбор МКАБ из существующих и заполнение номера ЭЛН.

{{ec02268f-cb9a-472c-95fa-7e6d057a27f0.png}}
                                                         
Для создания нового листка нетрудоспособности нажмите кнопку «Создать лист нетрудоспособности».

{{37f010cf-2a1e-4a93-9296-1efd818a9ff1.png}}

Листки нетрудоспособности можно оформить не только со страницы «Учет листков нетрудоспособности», но и из МКАБ, и из ТАП  (описано в ТМ_МИС_SaaS_Том_1).

h2.                            Создание листка нетрудоспособности из МКАБ

Создать новый листок нетрудоспособности можно из МКАБ пациента. Для формирования списка МКАБ выберете на главной странице пункт «Медицинские карты».

{{c2f7484e-8575-4b5d-89f3-969d6868c836.png}}

«Медицинские карты»
h1. Важно!* Если ЛН выдается по уходу за больным ребёнком или родственником, то в списке медицинских карт амбулаторного больного следует выбрать карту именно больного ребёнка/родственника.
В его карте должен быть обязательно указан представитель - родитель/родственник, на которого оформляется ЛН (получатель ЛН).
МКАБ пациента, которому необходимо создать ЛН, необходимо открыть и отредактировать.
Открытие карты осуществляется выбором пункта «Редактировать» в меню, открываемом нажатием кнопки или нажатием правой кнопки мыши по строке с МКАБ.

{{4e45c780-bc9c-4650-a0f1-bc311588b8a6.png}}                                                    

Для создания ЛН необходимо перейти во вкладку «Действия» выбрать пункт «Листки нетрудоспособности».                                                       . 
Пункт «Листки нетрудоспособности» позволяет создать для данного пациента листок нетрудоспособности или отредактировать уже существующий.
Если в МКАБ не заполнены обязательные поля или допущены ошибки, то появится соответствующее предупреждение, следует устранить ошибку и повторно открыть ЛН.
 
{{a0c51da8-4c88-416d-a5cf-f7924a53c0a7.png}}
 
{{fe508e94-ab3e-47ff-a1c0-18e9df9e377d.png}}
 
h3. Заполнение листка нетрудоспособности

Окно «Листок нетрудоспособности» состоит из пяти вкладок: (Общие данные), (Дополнительная информация), (Периоды/Нарушение/Закрытие),
(Журнал обмена с ФСС) и (Настройки).

{{f453d9c9-3652-422a-aa61-b387466021fd.png}}

В нижней части окна «Листок нетрудоспособности» расположены кнопки, доступные на любой вкладке:
(Испорчен), (Продолжение), (Совместительство), (Печать), (Сохранить), (Отмена).
Кнопка (Испорчен) используется для аннулирования испорченного ЛН и создания его дубликата  
Кнопка (Продолжение) используется для закрытия ЛН и создания ЛН-продолжения 
Кнопка (Совместительство) используется для создания ЛН по совместительству 
При помощи кнопки (Печать) можно распечатать талон нетрудоспособности и листок нетрудоспособности. Печать и калибровка ЛН перед печатью. 
Кнопка «Сохранить» на форме ЛН позволяет сохранить внесенные в ЛН изменения. 
Кнопка «Отмена» позволяет закрыть окно «Листок нетрудоспособности» без сохранения изменений. 

Область полей с данными ЛН Для оформления первичного ЛН  установите флажок «Первичный» (устанавливается по умолчанию), кнопка "Получения пула номеров из ФСС" предназначена для работы при оформлении электронного листка нетрудоспособности, возможно
ввести  номер ЛН с клавиатуры, указанный на бумажном бланке, и указать дату выдачи ЛН. После ручного заполнения поля «Номер» кнопка получения пула номеров станет неактивной 

{{338780d5-2438-42e1-8257-72fc07b56786.png}}

Неактивная кнопка получения номера из ФСС после ручного заполнения поля «Номер»
Поле «Номер» обязательно для заполнения
Если оформляемый ЛН является продолжением другого ЛН, то установите флажок «Продолж. листка №» и указать номер первичного ЛН.
Поле «Продолж. листка №» может быть заполнено автоматически, если в первичном ЛН на вкладке «Периоды/ Нарушение/ Закрытие» было заполнено поле «Номер следующего ЛН»;
поле также можно заполнить вручную с клавиатуры.
{{a61e3547-2a2b-452e-94fb-a905437e5d5e.png}}

Поле «Место работы» заполняется из справочника. Также, возможно добавление места работы в справочник через поле «Место работы», если организация отсутствует в данном справочнике.
Для добавления места работы в справочник и проверки его добавления требуется в создаваемом ЛН выполнить следующие действия:
В поле «Место работы» ввести наименование организации.
{{5dc82969-41d7-4efc-a144-172914b8d936.png}}                         
                      
Если указанное место работы является основным, то установите флажок «Основное место работы» (устанавливается по умолчанию).
Если ЛН выдается не по основному месту работы, то следует установить флажок напротив пункта «Совместительство»,
станет активно поле «№ по основному месту работы», в которое следует ввести с клавиатуры номер листка нетрудоспособности по основному месту работы.

{{e4a3705b-1003-458e-8ee0-329a869d625b.png}}                                     
                                
h3. Вкладка «Дополнительная информация»
 
На вкладке «Дополнительная информация» указывается информация о путевке в санаторий, лечении в стационаре, освидетельствовании с МСЭ и данных медицинской организации, выдающей листок нетрудоспособности.

В области полей об освидетельствовании в бюро МСЭ указывается информация о группе инвалидности пациента: дата направления в бюро МСЭ, дата регистрации в бюро МСЭ, дата освидетельствования в бюро МСЭ. 
{{871fb632-4e92-4561-9aa8-2d5b3be1605b.png}}
Можно указать группу инвалидности, выбрав её из выпадающего списка, открываемого установкой курсора в поле. Подробная инструкция есть в (Инструкция по работе с листками нетрудоспособности 1, 4.1.2 с. 22. Вкладка «Дополнительная информация»)
{{eded06a4-ad28-4c82-bf81-a19485a9af72.png}}
              
h3. Вкладка «Периоды/Нарушение/Закрытие»

На вкладке «Периоды / Нарушение /Закрытие» указывается информация о периодах нетрудоспособности пациента, нарушениях режима лечения и закрытии ЛН.
                                         
На вкладке можно добавить до трёх периодов лечения. По умолчанию отображаются поля для ввода данных по первому периоду. Поля «Первый период с» и «по» заполняются вручную с клавиатуры или выбором нужных дат в выпадающих календарях, появляющихся при установке курсора мыши в поле. 
Поле «количество дней» заполняется автоматически, при редактировании значения поля автоматически меняется дата окончания периода.
{{8bfb264e-0983-418f-9e3a-bc2b6226ede9.png}}
В случае если период нетрудоспособности более 15 дней, то необходимо заполнить ФИО и должность председателя ВК, иначе возникнет ошибка. Исключением является причина нетрудоспособности – «05 – Отпуск по беременности и родам».

{{681123ae-4542-4a10-be5f-13e4f64fcd1d.png}}
                      
Для добавления второго периода необходимо нажать {{749aefdf-5bea-48e8-a783-e69798c507f7.png}}  кнопку, расположенную под областью с данными о первом периоде.

{{e643483b-e9ab-43ca-acea-1674c02a18ca.png}}
                                                 
Период можно удалить. Для это следует нажать кнопку {{51c710bd-29fa-477e-b0a4-b593a2c389fe.png}}, расположенную над областью с полями о периоде.

{{de4f0c6f-dd9a-42e9-93c7-c0eb53457001.png}}

После заполнения поля «Иное» необходимо заполнить поле «Дата "Иное"». Для причин «31- продолжает болеть» и «37- направлен на долечивание непосредственно после стационарного лечения» поле Дата "Иное"» недоступно.

{{1d536fde-8c55-490c-bab8-40548201880a.png}}  
                                      
h3. Вкладка «Настройки»

Настройки «Запрос отправки ЭЛН в ФСС после подписания блока» и «Количество запрашиваемых номеров ЭЛН» предусмотрены для электронного листа нетрудоспособности.

{{2ffcfbc1-36f0-429b-8c11-ec5a5bb2371f.png}}

Для запроса пула номеров ЭЛН необходимо в поле "Количество запрашиваемых номеров ЭЛН" указать нужное количество номеров и нажать "Сохранить"

{{6fe76837-4c97-4a4d-8353-171d64922669.png}}

При нажатии на кнопку  {{d528d7d5-55d8-47a9-a5f0-8c247c6768b2.png}} программа присваивает свободные номера из ранее запрошенного пула номеров от ФСС. Если номера закончатся, то произойдет запрос в ФСС на количество свободных номеров указанное в настройке.

{{678fd828-1aea-4428-941d-c464532c9607.png}}

Для печати на ЛН штрих-кода должен быть установлен флажок «Выводить на печать штрих-код ЛН». При снятом флажке штрих-код не будет печататься. Настройка осуществляются отдельно для каждого ЛН.

{{ceea8e21-9726-4129-9923-eaf90d7812b5.png}} 
                                                           
В нижней части окна «Листок нетрудоспособности» расположены кнопки, доступные на любой вкладке:
«Испорчен», «Продолжение», «Совместительство», «Печать», «Сохранить», «Отмена».
Кнопка {{40b8eb1e-d913-416a-8bf9-bbf656ee9991.png}}   нужна для аннулирования испорченного ЭЛН и создания его дубликата

талон нетрудоспособности и официальный печатный бланк электронного листка нетрудоспособности, как целиком, так и по периодам.

h3.                                    Вкладка «Журнал обмена с ФСС»

На вкладке (Журнал обмена с ФСС) отображается информация об успешной отправке ЛН, а также ошибки, получения ФСС после отправки подписанных данных.
{{35c0e5ec-bce3-4691-94fb-818d75a2c18f.png}}
 
На вкладке «Журнал обмена с ФСС» можно вручную отправлять измененные данные в ФСС,
если была включена настройка «Не отправлять» или «Запрос перед отправкой в ФСС» (настройки описаны в Инструкция_по_работе_с_листками_нетрудоспособности_1)и если при запросе была нажата кнопка «Нет». Для отправки данных в ФСС необходимо нажать кнопку  {{dd664bf6-f3db-4605-bb24-804c8f99449e.png}}, которая появляется на вкладке после первой подписи данных в ЭЛН. Данные будут отправлены.


h3.                                   Создание ЭЛН из ТАП/МКСБ и массовое получение ЭЛН по пациенту

В ТАП/МКСБ доступен раздел «Листки нетрудоспособности». В разделе «Листки нетрудоспособности» ТАП отображаются ЛН пациента: открытые и закрытые, которые имеются в базе данных медицинской организации
{{29f6b976-5490-4588-afc9-0aba72b12f74.jpg}}

Для просмотре ЛН нажмите «Просмотреть» {{6a3518e7-2511-47d0-bc71-2382d31602c6.jpg}} , произойдёт переход в Зелёную МИС для просмотра ЛН, с возможностью редактирования.
Во вкладке «Листки нетрудоспособности» отображается полная информация по ЛН:
    * Номер ЛН
    * Медицинская организация открывшая ЛН
    * Дата открытия ЛН
    * Дата закрытия ЛН
    * Диагноз
    * Состояние ЛН
    * Количество дней
    * Количество периодов ЛН.

При выборе «ЛН по случаю лечения», отображаются ЛН, которые открыты в рамках данного случая
{{b7a0ecb4-9eb5-477d-9a65-610c290ae606.jpg}}

При нажатии на кнопку {{43fd15b0-477a-4502-a0b3-688a679e085c.jpg}} происходит переход на вкладку «Данные из ФСС». Здесь отображаются те ЛН, которые имеются в ФСС и не загружены в локальную базу данный медицинской организации. 
 Выгрузка ЭЛН из ФСС доступна только в ТАП
{{76ffc2f0-93f2-4b52-a8e5-e9abf8a415ca.jpg}}

При нажатии кнопки {{d5a23942-e248-4495-9d6c-5c08ae832192.jpg}}, данные будут загружены в локальную базу данных.

Для создания нового ЭЛН из ТАП нажмите кнопку {{7365ca70-f061-4b0f-9189-ec4bcfb3fa15.jpg}}, произойдет переход в зеленую МИС. Далее следуйте инструкции по созданию ЭЛН из МКАБ.
Если у пациента уже имеется один закрытый ЭЛН, который требуется продлить, нажмите на «Продлить» {{25e6be16-aa52-4bf0-ba4a-dda287fdd8d9.jpg}},, также произойдет переход в зелёную МИС с уже предзаполненными данными в ЛН. Далее следуйте инструкции по продлению ЭЛН.
Если у пациента есть открытый в медицинской организации ЛН, то будет доступна кнопка «Редактировать» {{22f87610-e969-4960-a3fd-061fdf12c774.jpg}}. Нажав на нее произойдёт переход в Зелёную МИС для редактирования
{{92ee1545-75b0-4a10-aeef-ee6761eea059.jpg}}

Для создания ЭЛН из МКСБ нажмите кнопку {{7365ca70-f061-4b0f-9189-ec4bcfb3fa15.jpg}}, произойдёт переход в Зелёную МИС. Далее следуйте инструкции по созданию ЛН.

h3.                                     Отчёты

Для формирования отчётов у врача должна быть роль *Отчёты в Веб-МИС*.

{{41158fa0-97a2-4bed-8d06-024e084d5544.png}}

Чтобы сформировать отчёт по *Листкам нетрудоспособности* нажмите на вкладку *Отчёты*.

{{49e87985-857a-4958-baf5-6680080a65d1.png}}

В поле поиск введите наименование отчёта.

{{6798dff1-f40b-430b-91d5-69add4bf377e.jpg}}

Выберите период формирования отчёта, нажмите *выполнить* .

{{9d4fe1a8-646d-4ee6-ba00-2597676af8db.png}}

После успешного формирования доступны несколько форматов для предпросмотра тела отчёта, *скачать файл *(скачивание файла в формате Excel), *Предосмотр PDF* (предпросмотр файла в формате PDF, позволяет распечатать отчёт), *скачать PDF* (скачивает файл в формате PDF), *Вернуться* (возвращает на страницу поиска отчёта).

{{3c73ab05-069f-4389-96d7-72c1c0e411f1.png}}

Выберите отчёт *Листки нетрудоспособности*. 

{{156dff95-3958-4b9d-ab87-015ec3e5e49f.png}}

При необходимости заполните фильтр формы.
