WEB Поликлиника: Журнал направлений

h1. Журнал направлений


h2. Настройка отображения

Зайдите в раздел "Администрирование", и выберете пункт "Системные настройки"

{{7fe8b90d-b3b5-4bda-ac93-358a7fb6dedf.png}}

Найдите и откройте настройку "Журнал Направлений Белый МИС", если настройки нет, то создайте ее. Появиться окно с настройкой, заполните следующие поля:
h1.  Значение, параметр - 1 (включение настройки), параметр - 0 (выключение настройки)
* Тип данных, Строковый 
* Пользователь, Admin (для общей настройки у всех пользователей)

После заполнения нажмите кнопку "Сохранить"

{{e6a75c0f-ae0f-4eed-abfa-c318d885b95b.png}}

h2. Роль

Для работы с Журналом направлений назначьте пользователю роль «Врач»

h2. Работа в Журнале направлений

Работа с модулем "Журнал направлений" осуществляется в блоке "Управление потоками пациентов". Для открытия модуля выберите на главной странице пункт "Журнал направлений"

{{b403b613-078f-4729-977d-d72f2d5af0e8.png}}

В результате перехода в новой вкладке браузера откроется "Журнал направлений"

{{daf7ba64-1a73-4520-ad98-7d10e68b1dde.png}}

Поиск направлений осуществляется по ФИО и полису, номеру, типу и даты направления, виду оплаты, статусу, направившему врачу, ЛПУ или МО другого региона, записи на приём и дате записи. Поля заполняются в ручную, выбором значения из справочника или установкой флажка.
*Поле "Дата записи" -  доступно для заполнения, если в поле «Запись на прием» указано значение «С записью»*

После того, как заполните поля фильтра, осуществите поиск направлений, нажмите клавишу Enter или кнопку "Найти". Отобразиться список направлений в соответствии с заданными условиями.
Для очистки всех полей фильтра и внести в них новые данные, нажмите кнопку "Очистить", все введенные данные будут удалены из полей

{{9afae987-284a-4dae-88f0-58a8c88ee7a5.png}}

Нажмите на кнопку "Сохранить в Excel", для выгрузки всех найденных направлений, файл выгружается в формате xls. Выгрузить можно не более 500 записей, иначе отобразиться предупреждающее сообщение

{{58720c88-2b4e-4856-90c7-b10ae56307fe.png}}

Список направлений представлен в виде таблицы, где указан номер, дата, ФИО пациента, куда направлен, тип направления, запись на прием, статус

{{d01faf68-701f-4d8f-b1d1-cce413614fa7.png}}

Для редактирования направления нажмите кнопку {{1f4a4d99-7131-4ef1-83c3-b173c70f2105.png}} в результате осуществится переход на форму редактирования направления. Кнопка «Редактировать» доступна для направлений в статусе «Черновик.

Для просмотра направления нажмите кнопку {{02f61885-5f92-415e-beaa-b12cce92c003.png}} Кнопка «Просмотр» доступна для направлений в статусах «Выписано», «Частично завершено», «Обработано», «Завершено», «Аннулировано», «Отказ», «Зарегистрировано».

Для печати направления нажмите кнопку {{9a1b613b-b0c8-41fd-bcb8-1b52b14d2027.png}} отобразится список печатных форм по выбранному направлению. Действие доступно для направлений в статусе «Выписано» и «Завершено».

Для перехода в случай лечения или аннулирования направления, нажмите кнопку {{f1b0db19-46f9-4162-929c-d547569f4ea8.png}} отобразится список дополнительных действий

{{a7045abb-e5a5-4b4f-91e7-10208390b042.png}}

{{173d4a96-4410-4b55-8529-11ebcbc1519a.png}}

Для направлений со статусом «Аннулировано» доступна возможность просмотра причины аннулирования. Наведите на иконку {{eb5a0976-5b12-4b84-a6a0-a8ed811b433e.png}} отобразится информация о причине аннулирования направления

{{4727309a-9530-4584-bdbe-6c06ab63111b.png}}

Аннулирование направления также доступно по кнопке {{8de982c2-e022-44b2-84c6-8e7f5be5b6e0.png}}
