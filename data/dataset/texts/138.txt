Правильное оформление и отправка СЭМД: СЭМД (230)"Медицинское заключение по результатам медицинского осмотра работника для предоставления в подсистему ЭЛМК"

h1. СЭМД (230) «Медицинское заключение по результатам медицинского осмотра работника для предоставления в подсистему ЭЛМК»


Данный СЭМД заменяет 194 СЭМД "Медицинское заключение по результатам предварительного (периодического) медицинского осмотра (обследования)" редакция 2

h2. Подготовка данных о пациенте

Для успешного формирования СЭМД в МКАБ пациента должны быть заполнены сведения:
	- О документе, удостоверяющем личность пациента (тип документа, серия, номер, кем выдан документ, дата выдачи документа);
	- О полисе ОМС пациента (тип полиса ОМС, серия, номер полиса ОМС);
	- Адрес пациента;
	- ФИО пациента (отчество при наличии);
	- Пол пациента;
	- Дата рождения пациента;
	- Место работы пациента;
	- Профессия и должность пациента;
	- О вредных производственных факторах;
	- Договор оказания платных услуг (не всегда, подробнее смотрите ниже).

h1. Обязательность заполнения поля «Договор платных услуг» зависит от вида оплаты:*
* Если при закрытии ТАП вид оплаты - ОМС либо ДМС, то поле «Договор оказания платных услуг» не обязательно для заполнения;
{{197684c9-fbcf-4e1e-b6e1-63f60450c8af.png}}

* Если вид оплаты - «Платные услуги», то поле «Договор оказания платных услуг» обязательно для заполнения; 
{{673a026f-420b-4d03-81d3-bad66c5318e6.png}}

* Договор необходимо указывать в блоке «Заключение» после заполнения карты мероприятий; 
{{524b0135-37fb-496f-b098-1139103a524c.png}}

* При отсутствии договора система сообщит об ошибке. 

Подробнее о создании «Договора оказания платных услуг» изложено в "статье":https://sd.hostco.ru/projects/int_ck/wiki/%D0%9F%D0%BB%D0%B0%D1%82%D0%BD%D1%8B%D0%B9_%D0%BC%D0%BE%D0%B4%D1%83%D0%BB%D1%8C.

h2. Заполнение данных для СЭМД

Формирование СЭМД осуществляется с формы «Карта мероприятий» по медицинским обследованиям «29н Предварительный медицинский осмотр работников» и «29н Периодический медицинский осмотр работников».
Формирование «Карты мероприятий» медицинского осмотра описано в "статье":https://sd.hostco.ru/projects/amurmis/wiki/WEB_%D0%94%D0%B8%D1%81%D0%BF%D0%B0%D0%BD%D1%81%D0%B5%D1%80%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D1%8F#%D0%A4%D0%BE%D1%80%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5-%D0%BC%D0%B0%D1%80%D1%88%D1%80%D1%83%D1%82%D0%BD%D0%BE%D0%B3%D0%BE-%D0%BB%D0%B8%D1%81%D1%82%D0%B0-%D0%B8-%D0%B7%D0%B0%D0%BF%D0%B8%D1%81%D1%8C-%D0%B2-%D1%80%D0%B0%D1%81%D0%BF%D0%B8%D1%81%D0%B0%D0%BD%D0%B8%D0%B5.

*Внимание!* *При формировании карты мероприятий данного СЭМД не обязательно создавать расписание.* 

При формировании карты мероприятий необходимо в графе «План» выбрать «29н Предварительный медицинский осмотр работников» или «29н Периодический медицинский осмотр работников».
Из списка нужно выбрать одну модель из поля «(Код) Наименование».
{{d43c91e5-2846-4661-87d0-e524c4134888.png}}

*Внимание! Формирование СЭМД ЭЛМК доступно только  для следующих моделей пациента:*
-	«Работы, где имеется контакт с пищевыми продуктами в процессе их производства, хранения, транспортировки и реализации (в организациях пищевых и перерабатывающих отраслей промышленности, сельского хозяйства, пунктах, базах, складах хранения и реализации, в транспортных организациях, организациях торговли, общественного питания, на пищеблоках всех учреждений и организаций)»;
-	«Работы на водопроводных сооружениях, имеющие непосредственное отношение к подготовке воды, а также обслуживанию водопроводных сетей»;
-	«Работы в организациях, деятельность которых связана с воспитанием и обучением детей»;
-	«Работы в организациях, деятельность которых связана с коммунальным и бытовым обслуживанием населения».

*Если у работника два вида работ* , то для успешного формирования СЭМД *должны быть сформированы две отдельные карты мероприятий*.

После создания «Карты мероприятий» необходимо заполнить ее.

h3. Условия заполнения «Карты мероприятий»

1) Результаты всех назначенных мероприятий должны быть выполнены на 100%;

1.1) Указать ФИО и должность врача в мероприятиях:  
- Осмотр врачом-терапевтом;
- Осмотр врачом-оториноларингологом;
- Осмотр врачом-дерматовенерологом;
- Осмотр врачом- стоматологом.

*Важно!* Должности врачей должны соответствовать мероприятию. *Например, осмотр врачом-стоматологом должен выполнить только врач с должностью «Стоматолог».*

h2. Просмотр и подписание СЭМД

Проверка документа перед его подписанием и отправкой в РЭМД производится по кнопке кнопке «Печать/ЭМД» с выбором пункта «СЭМД «Медицинское заключение по результатам медицинского осмотра работника для предоставления в подсистему ЭЛМК».
{{c13a6c84-9043-4d28-89b6-132076f3568e.png}}

В форме «Предварительного просмотра ЭМД» проверяются внесенные  данные. 
{{ff7ad0e9-73ac-42a5-8fde-9a70a4e47cb8.png}}

Формирование СЭМД «Медицинское заключение по результатам медицинского осмотра работника для предоставления в подсистему ЭЛМК» осуществляется при нажатии кнопки {{1155446f-2c0f-4fa3-bc8c-4654f8975c4d.png}} на форме заполненной карты мероприятий или форме предварительного просмотра ЭМД, открываемого по кнопке «Печать/ЭМД» – «СЭМД «Медицинское заключение по результатам медицинского осмотра работника для предоставления в подсистему ЭЛМК».

После подписания документа в правом верхнем углу экрана появится сообщение об успешной отправке в РЭМД. 
{{ca39dda8-7fdb-43d3-9bb4-a035155eb162.png}}

Печать документа для выдачи его пациенту либо сохранение в PDF производится выбором документа из списка по кнопке «Печать/ЭМД».
{{c13a6c84-9043-4d28-89b6-132076f3568e.png}}

На экране откроется окно печати документа и отобразится сама печатная форма.
{{53ff6cde-f115-4ea7-b376-f5006a1de2d5.png}}

h2. Оформление карты задним числом

Для оформления карты задним числом воспользуйтесь отчетом "Протоколы - Дата открытия карты ПМО". Более подробно с работой отчета можно ознакомиться по"ссылке":https://sd.hostco.ru/projects/amurmis/wiki/Дата_открытия_карты_ПМО.
При смене даты открытия карты, так же будет обновлена "Дата следующего ПМО" на дату ровно через год от даты открытия карты. 

*Изменения отображаются в форме СЭМД.*

h2. Возможные ошибки при формировании СЭМД
