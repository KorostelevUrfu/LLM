Лабораторно-диагностические исследования

h1. Лабораторно-диагностические исследования


h2. Описание



 предназначенный для автоматизации деятельности персонала лабораторий, позволяет вести учет направлений на исследования и их результатов.

h2. Функциональная диагностика

Для выписки направления на функциональную диагностику следует перейти в случай лечения, строка «Журнал направлений».  Выбрать тип направления «Лабораторные исследования» или «Диагностические исследования» нажать создать.

{{41158fa0-97a2-4bed-8d06-024e084d5544.png}}
{{85f426ac-4b3e-4942-96ff-0abba83ac427.png}}

В новом окне откроется форма заполнения «Направление на обследование» в статусе «Черновик».

{{e51f2ec9-6395-4d31-9376-6d67134e0ba2.png}}

Большая часть полей будет заполнена автоматически, нужно заполнить только «Цель исследования» и «Причину направления» выбрав вариант из списка. Номер направления не отображается, он присваивается автоматически после сохранения. Все поля за исключением «ТАП» и «Пациент» можно изменить.
Ниже находится блок «Исследования». Все поля данного блока являются обязательными, и заполняются выбором варианта из списка, порядок не важен. Для сокращения поиска исследований следует начать вводить значение с клавиатуры. При выборе «Исследования», поле «Вид исследования» заполнится автоматически. 

{{c8686e31-f8b9-4483-ac49-6c56ebdd07aa.png}}

После заполнения обязательно нажать «Добавить». После нажатия добавить, выбранные исследования отобразятся ниже.

{{90420a6c-ec42-4462-8637-da493159f392.png}}

После добавления всех необходимых исследований, направление нужно сохранить и подписать

{{ab39eaf5-3576-42ef-b9bc-c0d2ed7259a4.png}}

После сохранения направление перейдет в статус «Выписано», откроются новые действия. С данным направлением будет доступно следующее: «Печать», «Аннулировать», «Записать на прием», «Подписать и отправить».

{{36262170-dee0-416c-9022-eb3228e0708d.png}}

Печать по форме №57 доступна при нажатии кнопки печать. Форма №057/у-04 утверждена приказом Минздравсоцразвития России от 22 ноября 2004 г. №255

{{6ff29431-e307-40c3-b2db-8fa6890050fd.png}}

Аннулировать можно по нажатию кнопки «Аннулировать» и выбрать причину аннулирования направления. 

{{ee1736c9-7d5d-4f0c-8d40-936bf0f01a34.png}}

При нажатии «Записать на прием» откроется дополнительная форма для заполнения состоящая из 5 шагов: 1 шаг «Выбор учреждения», 2 шаг «Выбор специализации», 3 шаг «Выбор врача», 4 шаг «Выбор времени», 5 шаг «Подтверждение записи». Каждый шаг открывается после заполнения предыдущего.

{{02006eac-ac5a-4168-8310-2f3f5f9ac728.png}}

h3. Создание пакетных направлений на исследования

Для создания направлений сразу на несколько исследований предусмотрена возможность создания пакета направлений с последующей выпиской и выводом на печать.

Для того, чтобы создать пакет направлений, следует перейти во вкладку «Создать пакет направлений». Пакеты создаются из имеющихся в карте отдельно созданных направлений.

{{1dc54f6d-1c01-4cb5-af4d-2038b1a08b74.jpg}}

Установить "флажок" в строке с необходимым направлением, затем нажать кнопку «Создать пакет направлений». 

{{748509cc-b7e8-4abc-be76-58df65d4a950.jpg}}

Создать пакет направлений можно только для направлений на исследования в статусе: «Выписано», «Частично завершено», «Завершено».

{{5ebf6e7a-10d2-411e-bef2-7b6f17068a54.jpg}}

В результате откроется форма формирования пакета направлений, которую необходимо заполнить.

{{b59aa488-ee34-48e6-8377-1d3cb1eef1ab.jpg}}

Поле «Пользователь» заполняется автоматически данными пользователя, создающего пакет направлений, недоступно для редактирования. 

{{f71583c1-2a20-4ac2-bf47-dd2b5eebf2fa.jpg}}

В поле «Наименование пакета» необходимо ввести название пакета направлений на исследования вручную с клавиатуры, поле является обязательным для заполнения.

{{93152f38-ed69-4e2d-98f9-b07966198816.jpg}}

Если для пакета направлений нужно указать специальность, то следует установить "флажок" в поле «Пакет направлений для специальности».

{{16f5d3e8-2f6b-4c35-adeb-e855e1e6625c.jpg}}

В результате станет доступно поле «Специальность», в котором следует выбрать необходимую специальность из справочника.

{{29fea2aa-affe-4f47-ab30-82e4fbe9e912.jpg}}

Если при формировании пакета направлений специальность не была указана, то такой пакет направлений будет отображаться для всех пользователей. 
Если специальность была указана, то данный пакет направлений отображается для пользователей с указанной специальностью. 

В блоке «Направления для создания пакета направлений» отображаются исследования, выбранные для формирования пакета направлений.

{{f99253d7-548f-4dc6-b814-ce1dbe6d0f3e.jpg}}

Справа от наименования блока отображается общее количество выбранных направлений.

{{e3e07c5a-42fc-462e-b15a-533221914b67.jpg}}

Для просмотра списка исследований, входящих в направление, следует нажать кнопку рядом с нужным направлением.

{{ca1a89d6-0a30-4c59-96d0-22f03acaaec2.jpg}}

Список исследований представлен в виде таблицы со следующими столбцами: 
* «Вид исследования» – отображается вид исследования, входящего в направление;
* «Исследование» – отображается наименование исследования. 
Для скрытия списка исследований необходимо повторно нажать кнопку.

Если в направлении на исследование установлен признак «Срочное» , то в блоке «Направления для создания пакета направлений» рядом с таким направлением по умолчанию будет установлен флажок в столбце «Срочное».

{{f4adefa6-f7fb-4e12-8da3-033b3ed955cf.jpg}}

После заполнения необходимых данных следует нажать кнопку «Сформировать».

{{a521e7d8-a66b-4f7a-a20b-1d047ea1cddb.jpg}}

При успешном формировании появится соответствующее сообщение:

{{ccea5c76-7c25-49f2-942a-053ef94a3d53.jpg}}

Если наименование пакета направлений совпадает с наименованием существующего пакета направлений, то при формировании пакета направлений появится сообщение об ошибке. 
Необходимо изменить наименование и повторно нажать кнопку «Сформировать».

{{264ca1ed-d8a4-49fe-a3d4-4576676db069.jpg}}

Для направлений на исследования в статусе «Выписано», «Частично завершено», «Завершено», «Аннулировано» доступна кнопка «Развернуть», которая позволяет раскрыть направление для отображения входящих в него исследований. 
Список исследований представлен в виде таблицы со следующими столбцами:
− Вид исследования – отображается вид исследования, добавленный в данное направление;
− Исследование – отображается наименование исследования, добавленное в данное направление;
− Статус – отображается текущий статус исследования.

{{c0cc8200-9472-4fae-9f57-95725df4b263.jpg}}

Для создания направлений по сформированному раннее пакету следует перейти во вкладку «Выписать пакет направлений».

{{f7b72f71-73aa-4885-a03f-1a8d875fc958.jpg}}

Для выписки необходимо выбрать пакет направлений из выпадающего списка, открываемого щелчком мыши по нему, и нажать кнопку «Выписать». 
Справа от наименования пакета направлений отображается ФИО пользователя, создавшего данный пакет направлений.

{{89bdd1bf-3369-4f41-bf7c-46230eb3582d.jpg}}

Выписанные направления будут добавлены в список направлений в статусе «Черновик».

{{419df01f-1187-43e8-a6b7-1368d611e30d.jpg}}

Существует возможность добавления пакета направлений в «Избранное». Для этого необходимо рядом с нужным пакетом направлений нажать значок "звездочка".
После этого, рядом с «Избранными» пакетами направлений будет отображаться "активный" значок.

{{ecb8da1e-bea5-4445-a1b1-721386737cf3.jpg}}

Для просмотра «Избранных» пакетов направлений необходимо на форме выбора пакета направлений нажать на значок "звёздочка".

{{a45fc94d-28c7-42c9-9d03-6ce3443189b5.jpg}}

Отобразится список «Избранных» пакетов направлений.

{{70ed3b25-44e1-48d5-bf49-fca832497b33.jpg}}

Для удаления пакетов направлений из «Избранного», необходимо рядом с пакетом направлений нажать на значок "звёздочка".

Для удаления пакета направлений следует выбрать в поле «Выберите пакет направлений» нужное значение, затем нажать кнопку «Удалить». Кнопка «Удалить» доступна пользователю, создавшему выбранный пакет, и администратору Системы.

{{d5a4158f-4ad0-420e-b70f-57f4672044a8.jpg}}

В результате выполнения действия «Удалить» отобразится сообщение о необходимости подтверждения удаления пакета направлений.

{{adcba762-902d-496b-b41e-73972aad3c97.jpg}}

Для того чтобы подписать или редактировать выписанное направление следует нажать на значок "карандаш". Откроется заполненная форма «Направление на диагностические исследования» или «Направление на лабораторные исследования». 

Дальнейшая работа с направлениями на диагностические и лабораторные исследования представлена далее.

h3. Внесение результатов выполненного направления вручную

На главной странице Системы выбрать раздел «Направление на исследование»
{{undefined}}
В журнале направлений осуществляем поиск исследования, используя фильтр вносим данные для поиска и нажимаем кнопку "Найти".  В строке с исследованием  нажимаем {{undefined}}  или правой кнопкой мыши и выбираем пункт «Результаты исследования»  
{{156dff95-3958-4b9d-ab87-015ec3e5e49f.png}}

Также внесение результатов можно осуществить через редактирование исследований. Для этого следует в журнале направлений выбрать нужное и зайти в него нажатием на {{undefined}} 

{{4baee597-471b-44ef-8240-29a23eff0d7e.png}}

После чего откроется выписанное направление, в самом низу форму найти нужное исследование и нажать «Редактировать».

{{4ed793e9-4570-4ae9-9051-890781de253e.png}}

Откроется форма «Результат исследования». Верхняя часть формы будет заполнена автоматически данными из направления. Нужно нажать «Проведено» для открытия полей внесения результатов.

{{60521e10-7d42-4401-a994-3e49b7f8ce5e.png}}

Поля «Дата проведения», «Лаборант», будет заполнены автоматически текущими значениями, их можно изменить. Остальные поля заполняются вводом значения вручную (допускается копирование из других источников). После чего результаты нужно подписать, нажав {{undefined}} .
После подписания станет доступно «Добавить другое мнение», заполняется по аналогии с основными результатами. 

{{e5ab2188-7998-4667-a53e-79920d500540.png}}

Внесенные результаты можно распечатать, нажав «Печать» в нижней части формы.

{{fb52daad-4934-49af-b662-cc1da9f56c60.png}}

{{efef394c-3073-4c16-8151-21dbc5e8597c.png}}

h2. Ввод результатов исследований, если направление создано из другой МО

В случаи, когда запись на исследование в расписание была произведена другой медицинской организацией, есть возможность напрямую перейти в ввод результатов. 
В расписании, нажмите на слот записи левой кнопкой мыши, отобразится меню действий. Выберите пункт "Исследование". 

{{f1c92e44-fe06-4512-919a-d1a857d5d733.png}}

Отроется окно выбора направлений, выберите направление, которое было выписано в другой МО. Нажмите на {{1af87ad1-a724-499b-8817-c1ba029eada0.png}} и выберите действие "Ввод результатов исследования"

{{30836c10-45f9-47b2-8c98-4bd1dc95aa99.png}}

Откроется форма ввода результата. Для корректной работы, важно, создать Случай лечения. Нажмите кнопку "Создать случай лечения". Будет создан новый талон и строка заполнится данными о нем.

{{1036956c-5556-4cb2-b18f-6eb411b88a18.png}}

Далее проведите обычную работу с вводом результатов. 
ВАЖНО! Перед подписанием результат, проставьте чекбокс "Закрыть ТАП при подписании результата исследования", для автоматического закрытия созданного талона.
