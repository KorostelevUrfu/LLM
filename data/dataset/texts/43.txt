WEB Стационар: Гемотрансфузия: Работа с заявками и компонентами крови

h2. Работа с журналом заявок

Переход в блок "Журнал заявок на компоненты крови" осуществляется из подмодуля "Стационар", по средством выбора соответствующего пункта.

{{d50ca974-45f9-41de-a1c5-512375070393.png}}

Откроется Журнал заявок на компоненты крови, в котором осуществляется поиск по созданным заявкам, а также создаются новые. 
Для отбора срочных заявок, установите в фильтре флаг "Экстренно". Отобразятся только заявки со статусом "Экстренно"

{{6bbba1cf-5e3c-49ae-bead-dd5bc25549b3.png}}

Для отображения заявок, которые еще не были отработаны, установите фильтр по Статусу заявки на "Подписана" и нажмите кнопку "Найти". 

{{fcf58ee2-b852-45dc-b8ac-5ab8a4a56e15.png}}

Подробнее о статусах заявок:

h1.  Черновик - заявка в статусе черновик, были создана, но еще не подписана и не является актуальной. 
* Подписана - заявка выписана, подтверждена и готова к обработке. 
* Выполнена - заявка, по которой был добавлен компонент крови.
* Отклонена - заявка, по которой был оформлен отказ.
* Аннулирована - заявка, которую аннулировали. Заявка не актуальна. 

{{1dc05171-e98b-4a5f-9fc4-d6cee1233f6a.png}}

h2.  Работа с заявкой

Чтобы дать ответ на заявку нажмите на  {{7443b6b7-29f0-4cbc-b501-fb9ae98fdb0b.png}}, раскроется меню выбора
Доступно два варианта: добавить компонент и отказать в заявке

{{ff8cf6aa-d840-4b77-9aac-615e030a6db2.png}}

h3. Добавление компонента крови в заявку

В случаи одобрения заявки, выберете пункт "Добавить компонент"

{{57db4fed-6d73-4911-b285-916e6a84cd6e.png}}

Вас перенаправит в форму ввода информации о "Компоненте крови"
Укажите флаг "Перевести заявку в статус “Выполнено” после сохранения компонента", чтобы после добавления компонента крови, заявка отображалась, как "Выполнена"

{{a73ae4ac-f930-4611-9ed5-1745b451e76a.png}}

Заполните все обязательные поля, что выделены * (подчеркнуты на скриншоте). *Обязательным условием является добавление не менее 3х данных в блоке "Фенотип"* Для добавления нажмите кнопку "Добавить".

{{d565d84a-0501-4f17-8665-736717725435.png}}

Раскроется окно добавления. Заполните строку "Обозначение".
Все поля, за исключением интерпретации результатов, являются обязательными. После заполнения, нажмите кнопку "Сохранить"

{{e5d5e68d-a0bf-4ed5-88ba-e77ca8e5d2dd.png}}

После заполнения всех данных  о компоненте крови, нажмите кнопку "Сохранить" 

{{199c0d0a-52c0-4d1a-b630-64da1e1999f3.png}}

Если был включен флаг смены статуса, отобразится оповещение:

{{2d84c8d0-7390-4a27-8e51-853b2acebb85.png}}

h3. Отказ по заявке 

В случаи отказа, выберите пункт "Отказ"

{{7e432b0f-174a-4de3-9d37-ca0ff75437e9.png}}

Ручным вводом с клавиатуры укажите причину и нажмите кнопку "Отклонить"

{{a77bc739-f635-479f-8e24-5781ebaa53cf.png}}

Отобразится оповещение:

{{6bf24fb6-d40d-4618-b28c-69be7d74725b.png}}

И статус заявки в журнале будет изменен

{{08993059-3891-45a6-a944-db0d9954fa9d.png}}

h2. Работа с журналом компонентов крови

Переход в блок "Журнал компонентов крови" осуществляется из подмодуля "Стационар", по средством выбора соответствующего пункта.

{{b48db5fc-6ffd-43f1-b1bc-47e5421add0b.png}}

Откроется журнал компонентов крови.

{{b381b6cc-be22-4603-89f8-b567366f0196.png}}

Для поиска определенного компонента, задайте параметры и нажмите кнопку "Найти".

{{c7382131-09a8-460d-ac52-b5791705421c.png}}

h3. Добавить компонент крови без заявки

*Внимание! Добавленный без заявки компонент крови к заявке прикрепить нельзя, он будет отображаться, как свободный компонент в наличии, и использоваться только в "Протоколе гемотрансфузии"* 

Для добавления нового компонента, нажмите кнопку "Добавить компонент" в журнале компонентов крови.

{{f807f8b5-f645-49ed-8758-4a336e2794a7.png}}

Далее работа с компонентом, аналогична описанному алгоритму выше.
