WEB Регистратура: Создание и ведение расписания

h1. Создание расписания




h1. Роли необходимые для работы*

Для начала работы с функционалом регистратуры, добавьте сотруднику роль «Регистратура». Для получения доступа к функции

*ВНИМАНИЕ!* Роль «Врач» не совместима с ролью «Регистратура» при одновременном назначении роли «Регистратура» и «Врач» блокируется доступ к работе с ведением расписания приёма.

h3. Начало работы

Сотрудник должен выполнить вход в систему под своей учетной записью.
Для этого введите «Имя пользователя» и «Пароль» после чего нажмите кнопку «Войти»

{{41158fa0-97a2-4bed-8d06-024e084d5544.png}}

После входа в систему на главном экране отобразится интерфейс программы с доступными функциями. В зависимости от присвоенных ролей количество доступных функций будет меняться.

{{85f426ac-4b3e-4942-96ff-0abba83ac427.png}}

h3. Создание расписания

Для создания расписания выберите на главной странице функцию «Ведение расписания» {{8ff92fcc-17b9-4fc3-9881-e3ba14717ab1.png}}
Далее нужно выбрать один из типов ресурсов в выпадающем списке 

{{a596833e-b9fe-427b-9b66-eae77cb4b336.png}}

Пример создания расписания для врача:
1) В выпадающем списке выбрать тип ресурса «врачи»;
2) Выбрать врача из списка, для которого создаете расписание и отметить его флажком. (Для быстрого поиска можно воспользоваться строкой поиска, куда вводится фамилия врача, номер кабинета или наименование оборудования и т.д.
в зависимости от выбранного типа ресурса);

{{d6cd47c7-c13c-4176-a1b9-44c42965942b.png}}

3) Для создания расписания после выбора конкретного врача нажмите кнопку {{9c7debd3-c033-4428-9a0a-43e58f52cbc6.png}} после чего откроется окно создания расписания;

{{271740d9-cdea-47e9-9584-0f589101b312.png}}

4) В поле «Дата» укажите дату, на которую составляется расписание;
5) В поля «Начало интервала» «Окончание интервала» введите период времени, отведенный под определенный тип приема. В поле «Время приема» укажите время, отводимое на прием одного пациента;
6) Если поставить флажок в пункте «Без разбивки на интервалы», то время приема одного пациента не будет ограничено по времени. Если отметить флажком пункт «Живая очередь», то на указанный период времени возможно выдавать талоны прием, по которым будет осуществляться в порядке живой очереди. Так же станет доступно поле «Талонов в интервале», в котором укажите то количество талонов, приём по которым будет осуществляться в порядке живой очереди в этот период;
7) В поле «Тип приема» выбрать из выпадающего списка один из типов приема, который будет осуществляться врачом в указанный период. Пункт «Отсутствие приема» позволяет отразить в расписании, что в это время у врача нет прием и осуществить запись к этому врачу на этот период станет невозможно;
8) Далее следует указать права доступа к ячейке расписания. Доступны следующие права доступа к ячейкам расписания:
* Регистратура – записать пациента может пользователь Системы с ролью «Регистратура»;
* Врач – записать пациента может пользователь Системы с ролью «Врач»;
* Интернет – пациент может самостоятельно записаться к врачу через портал самозаписи;
* Другая МО – записать пациента может врач из другой медицинской организации по направлению;
* Инфомат – пациент может самостоятельно записаться через Инфомат к врачу;
* Кол-центр – записать пациента может оператор кол-центра;
9) После заполнения информации о интервале приема нажмите кнопку "Добавить" и созданный интервал расписания будет добавлен;
10) Для добавления новых интервалов повторите пункт 5-8;
11) Все добавленные промежутки расписания отобразятся в виде таблицы;
Промежутки можно удалять. Для этого выделите нужный промежуток и нажмите кнопку "Удалить". Для сохранения расписания нажмите кнопку «Сохранить».
Если при создании расписания установлен признак {{5aed4b13-b79a-452e-b810-6a5ddd1d338c.png}}, то расписание на выбранный день врача, будет заменено созданным. Об этом будет сказано в информационном сообщении,
которое появится при сохранении расписания. Если в существующем расписании имеются записи на прием, программа предложит их перенести.

Разные типы приема отражаются в расписании разными цветами.

{{8f3cc97b-0b76-488c-a0e0-cbd57b3fe1cc.png}}

ВНИМАНИЕ! Если у врача две и более должностей в разных отделениях, то для каждой должности заводится отдельное расписание. При сохранении расписания происходит проверка на существование
у данного сотрудника должностей в других отделениях и наличие у них расписания. При пересечении расписания появится сообщение об ошибке.

Аналогичные действия осуществляются при создании расписания для кабинета, оборудования, бригад и комиссий.

h3. Создание расписания для оборудования

Оборудование и кабинет являются разными ресурсами для составления расписания.  
При создании расписания на оборудование нет необходимости создавать расписание на кабинет с этим оборудованием.

Для создания расписания оборудования нужно на главной странице Системы выбрать пункт «Ведение расписания». Откроется страница составления расписания

h3. Особенности создания расписания для ресурса бригады

При создании расписания в поле «Тип приёма» следует выбрать значение «Приём бригады». При нажатии кнопки «Текущее расписание» будет отображено текущее расписание выбранного ресурса, а также расписание всех ресурсов, задействованных в составе бригады. Для возврата к форме добавления промежутков расписания следует нажать кнопку «Вернуться».
{{35db916a-caa4-4896-b405-e9f9b48f0668.png}}
{{2e08307b-873b-44b6-8d10-aa18b9936530.png}}
При установленном признаке «Мобильный комплекс» отображается для заполнения блок «Дополнительная информация». 
Поле «Адрес расположения бригады» заполняется с помощью справочника ГАР. 
Поле «Примечание» заполняется вручную с клавиатуры.
{{4c1aaefd-4e74-4d85-8c5e-5af6327ecd22.png}}

Выполняются шаги с 1 по 7 "Создания расписания", далее следует указать права доступа - *Интернет*.

После заполнения информации нажмите кнопку "Добавить". Для сохранения расписания нажмите кнопку «Сохранить».
