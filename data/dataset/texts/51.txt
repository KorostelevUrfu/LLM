WEB Стационар: Пакет направлений на исследования

h1. Создание направлений на исследования

Для создания «Направлений на исследования» войдите в пациента по кнопке {{933d53e6-198a-4c6f-9f45-3d5a01996a15.png}}

И в  разделе, «Медицинских записях» нажмите кнопку  {{2eda4caf-4cf7-4648-afab-8e10033e7fe0.png}} 
и выберите шаблон «Направления на исследования»

{{7fa6dc8d-61b9-4c4e-874b-48ac93ef2793.png}}

В шаблоне укажите: 
1.	МО лаборатории (для поиска введите три первых буквы МО), 
2.	Комментарий (пока поле обязательное), 
3.	Исследование (воспользуйтесь поиском по слову) и нажмите галку

{{77a8f883-a386-4cb0-8b13-5454ae220c19.png}}

Для исследования укажите параметры и для удобства сверните его

{{69cb2ec2-9c14-484f-8626-9ed3f2a62fc7.png}}

Найдите следующее исследование по ключевому слову и так далее, пока весь пакет не будет сформирован.

Для создания направлений нажмите кнопку «Сохранить направление»

{{c18ec67c-0d87-4d5a-ba15-77695bfe7251.png}}

Программа выведет сообщение о создании Направлений на исследования. Нажмите кнопку «Ок»

{{5c3e7b0b-8870-408b-88bf-e978f87eed02.png}}

После чего удалите шаблон по кнопке «Корзины»

{{46fb3b27-2206-4f6a-ac4b-f8b6641a62e6.png}}

h1. Пакет направлений на исследования

Для пакетного создания направлений необходимо выбрать все необходимые исследования, установить необходимые параметры и нажать кнопку "Шаблоны", в открывшемся блоке внести название "пакета" направлений и нажать кнопку "Сохранить"
{{1340e663-9867-4f88-814c-b54efb30b664.png}}

В дальнейшем для отправки созданного пакета, при открытии шаблона «Направлений на исследования» нажимаем на кнопку "Шаблоны", в открывшемся блоке выбираем необходимый "пакет"  и нажимаем кнопку "Выбрать". Данные внесенных исследований отобразятся в форме направления, при необходимости можно внести изменения, добавить параметр или удалить одно из исследований, для отправки направлений нажимаем на кнопку "Сохранить направление".
В блоке "Шаблоны" для фильтрации списка можно выбрать "Мои исследования" или "Избранные"

{{f02bc318-eadb-4160-bbf6-2f40a3ec42bd.png}}
