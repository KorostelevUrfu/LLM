WEB Поликлиника: Врач поликлиники

h1. Врач поликлиники

h2. Вызов пациентов и их неявка

Для приёма пациентов врач поликлиники работает с расписанием 
{{1148cc74-2ad3-4d16-8cd6-91cd0e6de378.jpg}}

В расписании выберете пациента, которого вызываете к себе
{{c58f4f2c-34bc-47cf-b6b2-654f09c63c70.jpg}}

Левой кнопкой мыши нажмите на пациента и выберете «Пригласить пациента»
{{099bd7d4-e97e-4c55-9ad5-249729e8fd9d.jpg}}

Откроется окно с началом приёма и отсчетом времени приёма пациента
{{1dfccd2e-e535-4a3f-a6d1-f0db3f10f269.jpg}}

В данном окне на панели есть 4 действующие кнопки: 
h1.  Прекратить вызов {{7d2ed089-1fe9-4e6e-a0c6-c5aecb749dbc.jpg}}
* Начать {{f570a03a-fc04-4fdb-b1d4-230b64f1534c.jpg}}
* Неявка пациента {{1e334a24-8ab2-4676-ba9a-88ee365dd3dc.jpg}}
* Следующий пациент {{ea849c6f-600c-49f2-adee-ca46f4890b76.jpg}}

Также имеется информация о пациенте, которого вызвали на приём. Чтобы начать приём нажмите на «Начать» {{f570a03a-fc04-4fdb-b1d4-230b64f1534c.jpg}}. Отсчет времени начнется по новой, а пациент в расписании будет выделен розовым цветом
{{69cb7035-0cdc-4728-9bfd-66d59be01f65.jpg}}

Для завершения вызова по пациенту нажмите {{f21fbabe-a669-496d-a3d5-7be6f7b3dcf8.jpg}} и завершите приём. До завершения приема по одному пациенту, можно вызвать второго, для этого нажмите кнопку {{ea849c6f-600c-49f2-adee-ca46f4890b76.jpg}}. Отобразится второй пациент в вызове, при этом предыдущий вызов не завершиться, а второй пациент, которого вызвали будет выделен зелёным цветом.
{{e253d897-8f92-4dfb-a3b4-29139d234382.jpg}}

Если пациент не явился, то нажмите на {{1e334a24-8ab2-4676-ba9a-88ee365dd3dc.jpg}}. Программа выдаст предупреждение, что далее с записью работать будет невозможно
{{acae9460-95e9-477a-907a-8f565b53ca8e.jpg}}

Нажмите кнопку  "Да", чтобы аннулировать запись, либо кнопку "Нет", если пациент явился.
При подтверждении неявки пациента в расписании его цвет изменится на серый, а статус изменится на «Неявка пациента»
{{d473a2bc-61fe-4fdc-a062-26bf3a4d1db3.jpg}}

Неявка также отмечается без вызова пациента. Для этого выберете пациента, который не пришел на приём, нажмите левой кнопкой мыши на него и выберете пункт "Неявка пациента"
{{d39ba312-92a3-4321-8744-264716aa2f00.jpg}}

Программа выдаст уведомление. Подтвердите неявку пациента
{{acae9460-95e9-477a-907a-8f565b53ca8e.jpg}}

Статус пациента изменился на "Неявка пациента", а пациент в расписании стал выделятся серым цветом
{{d473a2bc-61fe-4fdc-a062-26bf3a4d1db3.jpg}}

Перейти в ТАП вызванного пациента можно двумя способами:
1. Левой кнопкой мыши нажмите на пациента в расписании и выберете «Случай лечения»
{{43e17911-53f4-44c4-b12a-3d072cc52a55.jpg}}

2. Нажмите на номер, расположенный рядом со временем приема на панели 
{{06a31865-1b68-4397-954c-ee84eb14cefe.jpg}}

В новом окне откроется ТАП.

В разделе «Осмотр» заполняется Анамнез, для начала заполнение нажмите {{9311dca9-5122-4819-9390-d20f3be47002.png}}  в появившейся строке для редактирования или вода данных нажмите {{24faac53-77f4-40e3-8cb0-ec1ed6ecfec3.png}}.

{{e51f2ec9-6395-4d31-9376-6d67134e0ba2.png}}

Также есть возможность добавить патологические реакции. Нажмите на стрелочку и кнопку «Добавить»
{{4a644bba-540b-494d-b7e6-eafdae88fd15.jpg}}

Заполните реакцию и результат, врач заполнится автоматически, но выбирается также из выпадающего списка и нажмите «Сохранить»
{{9b61a476-f6a5-4f44-80c3-0419f98c951f.jpg}}

{{f2cc4297-1430-41ae-8f4e-0fadc2b584b1.jpg}}

Если у пациента есть вредные привычки, то внесите их, нажмите «Сохранить». Список подлежит редактированию
{{0ed49730-2e1a-4cfb-aba9-2ce68f5c3d98.jpg}}

h3. Показатели здоровья

В разделе "Показатели здоровья" отображаются ранее заполненные показатели здоровья. Заполняются "Показатель" и "Группа показателей", также их можно заполнить по случаю лечения. Заполнение происходит по аналогии с предыдущим разделом.

{{84856ae1-d09c-4ff8-8b4a-a7145822f287.png}}

Добавлена группа "Клинический минимум обследования", для заполнения необходимо выбрать группу и внести значения, данные отображаются по последнему сохраненному значению обследования пациента внесенных в вручную.
{{ed944c5b-a6bf-4a30-9d33-f38ef03ecf0e.png}}

h4. Заполнение  шаблона МЗ "Осмотр врача по стандарту"2.0

В шаблоне «Осмотр врача по стандарту 2.0»,  заполняются только те поля, которые Вам необходимы. В печатной форме незаполненные поля не отображаются. 
Используя функцию «Словарь» можно сохранить слова, фразы и предложения для последующего выбора и отражения в поле документа. Для отображения данных словаря под полем, необходимо нажать на кнопку "Закрепить"
{{9166132d-0dbb-4edb-8ce7-6c0eb341989a.png}}

Для внесение основного диагноза, нажимаем на кнопку «Основной». В открытом окне заполняем «Код по МКБ-10». Выбираем из выпадающего списка «Характер заболевания» и «Диспансерный учёт», при наличии описаний заполняем поле «Описание», нажимаем «Сохранить». 
Для добавления сопутствующего и др. диагнозов, необходимо нажать на кнопку "Добавить диагноз". Открывается окно для заполнения, с возможностью выбора из выпадающего списка «Тип диагноза».
{{3359c84f-4c60-442c-a4f1-0e55ce791339.png}}
Заполняем данные по оказанной услуге. Выбираем из выпадающего списка «Услугу» и «Диагноз», нажимаем «Сохранить». 
{{b1447083-7c83-4e18-b7fd-7cbdec01d653.png}}

h4. Функция "Заполнение по истории"

Заполнить медицинскую запись можно с помощью данных ранее внесенных в прошлые медицинские записи.
Для этого  в шаблоне №200 МЗ "Осмотр врача по стандарту 2.0" в верхней части меню выбрать значок  {{2e17ad96-ffae-425d-97c8-707548d1d5ff.png}} нажав на который откроется окно для выбора предыдущей оформленной мед.  записи.

{{2de9bb25-2837-4f6e-86ea-6363d67e2adf.png}}

Переносить из предыдущей медицинской записи в рамках данного случая лечения, даже если эту запись в нем оставил другой специалист.
Для переноса данных из предыдущей мед. записи следует включить чекбокс, тогда каждый раз при создание новой медицинской записи, автоматически будут переноситься значения из предыдущей. 

{{5e002539-b0ed-4e37-a1d1-df111b062543.png}}

Переносить из предыдущей медицинской записи предыдущего случая лечения. С условием, что пациенту выставлен один и тот же диагноз. Также не имеет значение какой врач оставил запись.
Чтобы перенос был осуществлен, прежде чем создать запись в новом случае, нужно сперва установить тот же диагноз, как в случаи из которого хотим осуществить перенос. Диагноз должен совпадать до точки.
Для переноса данных из предыдущего лечения следует включить чекбокс:

{{dd1b35af-dc0b-4c4a-879d-99526a026289.png}}

Для поиска медицинской записи необходимо включить чекбокс поликлиника либо стационар (где была оформлена прошлая мед. запись).
Выбрать временной промежуток и нажать кнопку поиска:

{{8bf14c5c-045b-458d-8eff-a9d90b5d7e32.png}}

Программа сформирует список подпадающих под заданный фильтр медицинских записей:

{{4da2d107-09f6-4796-aa76-0c5c60bda81c.png}}

После выбора медицинской записи появится поле с ранее заполненной информацией которая может быть перенесена в текущую медицинскую запись:

{{dfa4ed88-1b56-46b2-9888-04a5405af107.png}}

Сведения из документы заполненного ранее будут перенесены в текущую мед. запись:

{{8f0a98fe-32ac-4e2a-aa80-f69965763f7d.png}}

Для сохранения заполненного шаблона и применения в работе есть функция "Избранные шаблоны"  {{6eaf4e09-aee0-48c2-9f2b-8aef697d2ce5.png}}

h4. Функция "Избранные шаблоны" 

Для сохранения заполненного шаблона и применения в работе есть функция "Избранные шаблоны".

2. В верхней части меню выбрать значок   {{6eaf4e09-aee0-48c2-9f2b-8aef697d2ce5.png}} нажав на который откроется окно "Избранные шаблоны"

{{d68f04cc-2aa2-4527-8751-870dcd2dac7a.png}}

Шаблон создан и будет отображаться в списке при открытии любого нового шаблона МЗ:

{{ef4a375e-54b3-4cd5-9649-178657fdc04a.png}}

Для изменения названия шаблона следует  нажать кнопку  {{3d94e988-cdba-465b-ad73-b02333f5ed87.png}}

Для сохранения изменений следует нажать кнопку  {{5bf7b812-6eec-41f4-8348-5493cb213c2d.png}}

Для удаления  шаблона следует  нажать кнопку {{c3e73ca4-287b-4738-90f0-ed765cf44bc8.png}}

Для поиска шаблона следует ввести название либо включить чекбокс для вывода на экран всех созданных ранее шаблонов  {{b55c9a2f-d0cf-478b-8755-4d57333da24f.png}}

4. Для применения шаблона:
4.1 в не заполненной медицинской записи на панели меню нажать кнопку {{6eaf4e09-aee0-48c2-9f2b-8aef697d2ce5.png}}
4.2 найти созданный в п. 3 шаблон из списка и нажать кнопку  {{b8cfafdb-e66b-4167-b6ce-aacb9e0b77a9.png}}

h4. Функция "Направление на исследование"

Чтобы создать направление на исследование пациенту через шаблон №200 МЗ "Осмотр врача по стандарту 2.0" нужно в верхней части шаблона в меню выбрать значок {{e73fac08-16da-49ee-bdc3-06ec39af2c1c.png}}, нажав на который откроется окно направлений

Для назначения лекарственного препарата необходимо заполнить поля:

- Дата направления - по умолчанию текущая, с помощью календаря можно выбрать иную дату, но не менее текущей;
- Время -  по умолчанию время открытия окна направлений, можно изменить вручную;
- МО лаборатории - заполняется выбором значения из справочника;
- Цель исследования - заполняется выбором значения из справочника;
- Причина направления - заполняется выбором значения из справочника;
- Контингент - заполняется выбором значения из справочника;
- Тип оплаты - заполняется выбором значения из справочника;
- CITO - срочное направление - активируется при нажатии на чекбокс;
- Повторное направление - активируется при нажатии на чекбокс;
- Диагноз - автоматически отображается диагноз из Случая лечения, при необходимости можно выбрать иной из справочника;
- Комментарий - заполняется с помощью ранее сохраненных данный в Справочнике либо вручную;
- Описание причины - заполняется с помощью ранее сохраненных данный в Справочнике либо вручную.

h4. Функция "Назначение ЛН"

Чтобы назначить ЛС пациенту через шаблон №200 МЗ "Осмотр врача по стандарту 2.0" нужно в верхней части шаблона в меню выбрать значок {{31d843a0-02ac-49b2-a974-26dc6c27815d.png}}, нажав на который откроется окно назначений

Для назначения лекарственного препарата необходимо заполнить поля:

- Дата назначения - по умолчанию текущая, с помощью календаря можно выбрать иную дату, но не менее текущей;
- Время -  по умолчанию время открытия окна назначений, можно изменить вручную;
- Диагноз - автоматически отображается диагноз из Случая лечения, при необходимости можно выбрать иной из справочника;
- Препарат - заполняется выбором значения из справочника;
- МНН - заполняется автоматически;
- Начало исполнения - по умолчанию дата текущая, с помощью календаря можно выбрать иную дату, но не менее текущей;
- Дни приёма - заполняются вручную;
- Интервал - заполняется выбором значения из справочника;
- Разовая доза - автоматически отображается доза выбранного выше препарата;
- Ед. измерения - автоматически отображается ед. измерения выбранного выше препарата;
- Раз в день - количество раз приемов препарата в день заполняются вручную, по умолчанию стоит 1;
- Кол-во на курс - количество раз приемов препарата на курс заполняются вручную, по умолчанию стоит 1;
- Способ приёма - заполняется выбором значения из справочника;
- Путь введения - заполняется выбором значения из справочника;
- Детализация дозировки - автоматически формируется по совокупности данных из полей: "Способ приёма". "Разовая доза", "Путь введения", "Раз в день", "Дней приема";

h3. Сведения о врачебной комиссии

Отображение раздела «Сведения о врачебной комиссии» доступно для пользователей с ролью «Сведения о врачебной комиссии». 
В разделе «Сведения о врачебной комиссии» отображается блок «Направления на врачебную комиссию». Блок «Направления на врачебную комиссию» содержит вкладки «Решение ВК по текущему случаю» и «Все решения ВК пациента». Для создания направления на врачебную комиссию следует нажать кнопку «Добавить» на вкладке «Решение ВК по текущему случаю»
{{2be18832-4030-4f51-9beb-27fd87c86ac3.png}}
Заполняем обязательные поля направления на врачебную комиссию:
В поле «Тип консилиума» следует указать тип консилиума врачебной комиссии, заполняется путем выбора нужного значения из справочника.  Если в Случае лечения указан онкологический диагноз из диапазона кодов МКБ С00-C97 и D00-D09, то поле предзаполняется значением «Онкологический».
В поле «Врачебная комиссия» выбирается значение из справочника ресурсов с типом «Комиссия».
В поле «Цель заседания врачебной комиссии» указывается цель из справочника «Цель заседания ВК», по которой пациент направляется на ВК.
В поле «Характеристика случая экспертизы» указывается характеристика случая экспертизы из справочника, по которой пациент направляется на ВК.
В поле «Перечень обсуждаемых вопросов» следует ввести вручную с клавиатуры перечень вопросов для обсуждения врачебной комиссией.
В поле «Заседание врачебной комиссии» следует выбрать заседание врачебной комиссии. Если нет подходящего заседания ВК в выпадающем списке, то его можно создать с помощью кнопки «Создать заседание ВК». 
    - Поле «Дата» заполняется путём выбора даты из календаря или вводится вручную с клавиатуры. По умолчанию указана текущая дата.
    - Поле «Форма проведения» заполняется путём выбора нужного значения из выпадающего списка. По умолчанию указано значение «1 - Очно».
    - После заполнения полей следует нажать кнопку «Создать»
После заполнения поля «Заседание врачебной комиссии» следует нажать кнопку «Записать».
{{d63299cf-a08b-4bc5-bf64-458bc515bb7c.png}}
Для перехода в "Решение врачебной комиссии":https://sd.hostco.ru/projects/amurmis/wiki/СЭМД_%22Выписка_из_протокола_врачебной_комиссии%22#СЭМД-Выписка-из-протокола-врачебной-комиссии нажимаем на кнопку "Действия" выбираем "Перейти в решение ВК".

h3. Выписать назначение и сформировать рецепт

Для выписки рецепта необходимо наличие ролей "Врач", "Рецепты".
Для создания назначений и выписка рецептов необходимо нажать на кнопку "Выписать рецепт".
В новой вкладке откроется форма «Назначения», на которой при наличии отображаются имеющиеся назначения пациента.
Для добавления нового назначения необходимо нажать кнопку "Добавить" и выбрать тип назначения. Откроется форма ввода данных нового назначения.
*Поля помеченные символом* «*» *обязательны к заполнению, если поля не заполнены, то они будут выделены красным цветом.*
При нажатии кнопки «Сохранить» назначение сохранится со статусом «черновик» с возможностью редактирования. Для сохранения назначения необходимо нажать кнопку «Подписать».
{{5f72d17b-fffd-4cc6-885f-cbaf6bb85ede.png}}
После сохранения и подписания назначения становится доступна кнопка "Добавить" для создания рецепта по этому назначению. Откроется форма создания рецепта.
Флажок, установленный в поле «Бланк», позволяет ввести «серию» и «номер» рецепта в случае, если он выписан на типографском бланке.
Для выписки электронного рецепта необходимо установить признак «Электронный». Признак «Электронный» установлен по умолчанию при открытии формы. Для выписки электронного рецепта в МКАБ у пациента должен быть установлен флажок «Согласие на электронный рецепт».
Для сохранения данных рецепта нажмите кнопку «Сохранить». Для подписания бумажного рецепта нажмите кнопку «Подписать».
{{9a0d59ec-3706-4f45-bfa6-8073f117b6b5.png}}
После сохранения и подписания  рецепта становится доступна кнопка «Печать». Для печати оборотной стороны рецепта предварительно установите признак «Печать оборотной стороны».
